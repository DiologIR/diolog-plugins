# mac-design-studio

Design beautiful, varied, authentically native macOS app UIs and icons from scratch.

The generation counterpart to [`mac-design-digest`](../mac-design-digest): where digest *learns* taste from curated apps, studio *applies* it. Every design:

- starts from a **named aesthetic direction** chosen from a catalogue synthesised from 135 curated [macapp.supply](https://macapp.supply) apps (`references/design-directions.md`) or 134 icons (`references/icon-directions.md`)
- stands on the **macOS 27 platform floor** — the control ladder, SF Pro ramp, label tiers, and chrome metrics deconstructed from Apple's own UI kit (`references/native-foundation.md`)
- routes native correctness to the **macosify** HIG library and craft/variation/polish to **design-craft** and **ux-craft**
- ships only after a **10-point native-tells audit** + **14-point quality rubric**, an AI-slop check, and a lookalike check (inspiration, never cloning)
- obeys **variety discipline**: consecutive designs steer away from repeated directions, palettes, and glyph types

## Install

```
/plugin install mac-design-studio@diolog-plugins
```

Best alongside: `macosify`, `design-craft`, `ux-craft`, `mac-design-digest`.

## Use

```
> Design the main window for a macOS podcast editor — pro audience, dark mode
> Give me 3 different directions for a menu-bar time tracker
> Make an app icon for "Ledger", a personal finance app — calm, trustworthy
```
